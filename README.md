In this project I have created a efficient tool for directories and files scanning in Windows, generating detailed reports in plain text (`.txt`) and web pages (`.html`) format.

## Features

- **Modular scanner:** Optimized logic for analyze paths of the file system.
- **Multi-format Report:**
  - **TXT:** Ideal for quick reading from the console or system logs.
  - **HTML:** Visual and structured report for a better interpretation of the analyzed data.
- **Clean Code:** Separate architecture by responsibilities to facilitate script maintenance and scalability.

## Requirements

Ensure you have Python 3.x installed in your Windows system.
You also need to install the packages of the requirements file. You can do it with this command:
```powershell
pip install -r requirements.txt
```
